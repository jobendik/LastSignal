# LAST SIGNAL — Gameplay Guide

## What kind of game is this?

**LAST SIGNAL** is a tactical sci-fi **tower defense roguelite**. You defend the Signal Core from lane-based anomaly waves by placing and upgrading towers, deploying drones, and adapting your build between waves.

Runs are self-contained, but your account/profile progresses over time through codex discoveries, achievements, research points, and persistent unlocks.

---

## Core Goal

Your primary objective is simple:

- **Protect the Signal Core** until all sector waves are cleared.

You lose when:

- **Core Integrity reaches 0**.

You win when:

- You survive all designed waves for the selected sector and defeat final threats (including boss waves where applicable), which triggers **Victory / Signal Held**.

---

## High-Level Game Flow

A typical run follows this structure:

1. **Main Menu**
   - Review profile progress and previous runs.
   - Open Codex/Research/Settings.
   - Start a mission.

2. **Sector Select**
   - Pick a sector (map layout, spawners, and wave set).
   - Difficulty and loadout choices shape early strategy.

3. **Planning Phase (before each wave)**
   - Build new towers.
   - Upgrade/sell existing towers.
   - Buy/support drones.
   - Read wave intel (enemy composition + recommended counters).
   - Start the wave manually (or early for tempo/economy advantages).

4. **Wave Active**
   - Enemies spawn from active lanes and path toward the core.
   - Towers/drone systems engage automatically.
   - You react with emergency tools (EMP, repair timing, positioning decisions for future waves).

5. **Wave Complete / Reward Choice**
   - Earn credits and optional reward upgrades.
   - Rebuild and re-optimize for the next incoming pattern.

6. **End State**
   - **Victory** when all sector objectives are completed.
   - **Game Over** if the core falls.
   - Receive end-of-run stats and grade feedback.

---

## Gameplay Loop (Moment-to-Moment Decisions)

During a run, you constantly choose between short-term survival and long-term scaling:

- Spend now on immediate DPS/control, or save for better timing.
- Go wide (more towers) vs. go tall (upgrades/specialization).
- Invest in economy towers/harvesters vs. combat pressure.
- Build to counter the next wave’s abilities (phase/heal/armor/swarms/etc.).
- Preserve credits to retain emergency options (repairs/abilities).

This pressure curve is the central “skill expression” of LAST SIGNAL.

---

## Main Systems You Use to Play

### 1) Towers

Towers are your primary defense. Each tower type fills a role:

- **Direct DPS** (consistent damage)
- **Control** (slow/stun/zone denial)
- **AOE** (wave clear)
- **Utility/Economy** (resource scaling, support effects)

Effective runs combine roles rather than over-committing to one damage profile.

### 2) Drones

Drones provide mobile support and can fill tactical gaps across lanes. They are especially useful when threat shifts between spawners.

### 3) Economy

Credits are earned primarily by wave clears and kills. Credits are spent on:

- Building towers
- Upgrading towers
- Strategic recovery (repairs and run-stabilizing actions)

Economy discipline determines whether you can absorb difficult spikes.

### 4) Upgrades / Rewards

Between waves (or on reward screens), you pick upgrades that shape your run identity:

- Global damage/range tempo
- Niche anti-enemy tech
- Synergy multipliers for specific tower plans

### 5) Core Actions

The core has emergency interaction tools (for example, timed abilities/repairs depending on state). Strong players treat these as planned resources, not panic buttons.

### 6) Multi-Core Expansion (New)

As a run progresses, you can now deploy additional **Relay Core Nodes** on the map:

- Available in planning/wave-complete phases after early waves.
- Built as a 2x2 core cluster for a significant credit cost.
- Increase total core durability and expand your defensive network.
- Enemies path toward the nearest available core cells, so new core placement changes lane pressure.

---

## Combat & Lanes

Enemies spawn from designated spawners and follow computed paths toward the core. Your lane control plan should account for:

- Chokepoint value
- Range overlap between towers
- AOE coverage on high-traffic tiles
- Anti-breach fallback near the core

A stable layout usually has:

- **Outer pressure reduction** (early chip damage)
- **Mid-map control** (slow/AOE)
- **Inner fail-safe layer** (high reliability anti-breach)

---

## Wave Phases and Difficulty Curve

As wave count increases, pressure escalates through:

- Higher enemy volume
- Faster/stronger enemy stats
- Special traits and bosses
- Compounding composition complexity

The game expects adaptation: static one-plan setups eventually collapse.

---

## How to Win Consistently

1. **Respect wave intel** every round.
2. **Build mixed damage + control**, not pure greed.
3. **Keep an emergency credit buffer** when possible.
4. **Upgrade intentionally** (key towers first).
5. **Use planning phase fully**; don’t autopilot immediate starts.
6. **Protect core integrity in early/mid game** to preserve margin for late waves.

---

## Failure Conditions (Why runs are lost)

Common loss patterns:

- Over-greeding economy and under-defending early.
- No answer to a specific enemy ability spike.
- Poor lane overlap leading to leak cascades.
- Spending to zero and losing access to emergency stabilization.

---

## Progression Outside a Run

Meta systems provide long-term motivation:

- Codex entries discovered from enemy encounters.
- Achievement unlocks.
- Research point economy and persistent upgrades.
- Run history tracking (performance benchmarking).

This makes each defeat informative, not wasted.

---

## Controls (Typical)

Common hotkeys include:

- Number keys for tower selection/build slots.
- Upgrade / sell actions.
- Start wave.
- Pause and speed controls.
- Utility actions (e.g., drone or tactical functions).

Check the in-game settings panel for your current bindings.

---

## End of Run Feedback

At run end, you receive:

- Sector/wave progress summary
- Combat/economy performance stats
- A run grade to help evaluate consistency and efficiency

Use this screen to decide what to change on the next run (composition, spend pattern, lane priority, or timing).

---

## One-Sentence Summary

**LAST SIGNAL is won by balancing economy, lane control, and counter-building across escalating waves while preserving enough core integrity and resources to survive late-game pressure.**

---

## RTS-Style Expansion Layer (Warcraft-Inspired)

The game now includes stronger RTS-style base progression inspired by classic macro RTS loops:

- **Command Tiers (T1 → T3):** upgrade your command level in-run to unlock stronger military tempo.
- **Population Growth:** higher command tiers increase drone population cap.
- **Militia Waves:** at higher command tiers, your base periodically trains and launches temporary militia drone squads automatically during active combat.
- **Forward Base Play:** additional relay cores let you establish multi-core defensive territory and shape path pressure, similar to expanding to new bases.
