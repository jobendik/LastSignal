# LAST SIGNAL Privacy Notice

Last updated: May 11, 2026

LAST SIGNAL is designed to collect as little personal data as possible. The game
runs in your browser and stores normal progress, settings, and run history in
local browser storage.

## Data We Collect

LAST SIGNAL may record anonymous gameplay events so we can understand how the
game is performing and improve balance. Examples include sector progress, wave
results, upgrade choices, tower usage, completion outcomes, and similar
gameplay statistics. These events are not intended to identify you.

If you allow cloud save, LAST SIGNAL may save your progress through CrazyGames
cloud services. Cloud save data is tied to the CrazyGames user id provided by
CrazyGames, so your progress can sync when you play through that platform. Cloud
save may include campaign progress, research unlocks, achievements, settings,
and recent run history.

If you allow ads, LAST SIGNAL may request ads through the CrazyGames SDK.
CrazyGames and its advertising partners may process data needed to show and
measure ads according to their own policies.

## Data We Do Not Collect

LAST SIGNAL does not ask for or collect your email address. The game does not
store your IP address in its own systems. LAST SIGNAL does not include a
separate third-party analytics SDK. We do not sell personal information.

## Your Choices

On first load, LAST SIGNAL asks whether you want to accept all optional data
features or use essential-only mode. You can change this later in Settings under
Privacy & data. Essential-only mode disables ad requests and CrazyGames cloud
save permission. Local browser saves remain available so the game can function.

## How To Delete Your Data

To delete local data, clear the browser data for the site where you play LAST
SIGNAL. This removes local progress, settings, consent choices, and saved run
history from that browser.

To stop optional data features, open Settings, choose Privacy & data, and select
Essential only. If you used CrazyGames cloud save, also manage or delete your
CrazyGames account data through CrazyGames account or platform controls.
