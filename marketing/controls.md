# Last Signal — Controls Reference

## Desktop (Mouse + Keyboard)

| Action | Input |
|---|---|
| Place tower | Click empty grid node → select tower type in Build Menu |
| Select tower | Click placed tower |
| Upgrade tower | U (or click Upgrade button in Tower Panel) |
| Sell tower | S (or click Sell button in Tower Panel) |
| Relay deploy | R |
| Start wave | Space |
| Pause / Unpause | P |
| Speed up | + / = |
| Speed down | - |
| Quick-build tower 1–6 | 1 / 2 / 3 / 4 / 5 / 6 |
| Specialize tower | Y (toggle command tier) |
| Field Manual / Codex | H or ? |
| Wave preview | Tab |
| Evac squad | Q |
| Retask squad | E |
| Squad F1–F4 | F1 / F2 / F3 / F4 |
| ESC / close panel | Esc |

## Mobile (Touch)

| Action | Input |
|---|---|
| Place / select tower | Tap grid node |
| Build drawer | Tap bottom build bar or swipe up |
| Upgrade / Sell | Tap tower → action sheet slides up |
| Start wave | Tap LAUNCH button (top bar) |
| Core EMP | Tap EMP button (top bar, during active wave) |
| Pause | Tap pause icon (top right) |
| Pan camera | Drag |
| Zoom | Pinch |
| Open command menu | Tap hamburger icon (top bar) |
